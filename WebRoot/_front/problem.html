1. 新手上路问题

2014-12-19 应用试客

应用试客是一个应用体验平台，通过试玩账号内的每个应用可获得奖励 3元、2元不等。

目前 “试玩” 奖励仅针对 iOS（苹果）用户开放，且越狱设备无法试玩。安卓、iOS、越狱用户可以通过邀请 iOS好友试玩应用获得奖励。

试玩奖励

点击【开始赚钱】按照步骤引导进行身份验证后，即可开始试玩并且获得试玩奖励。

邀请奖励

获得的现金奖励，可通过绑定支付宝提现。设置路径：【我的信息】-【个人中心】。

每个关注应用试客的用户都有一个试客 ID。你的试客 ID 可以在【个人中心】查看。

请注意：小点儿绑定的试客 ID = 唯一设备 + 唯一微信 + 唯一苹果账号，不能混淆，即：

  • 试客1= 设备1+ 微信1+ 苹果账号1
  • 试客2= 设备2+ 微信2+ 苹果账号2
